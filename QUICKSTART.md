# Quick Start Guide - AutoFolder AI

## Installation

**No installation required!** AutoFolder AI is portable.

1. Download `AutoFolder_AI.exe` (or ZIP file)
2. Double-click to run
3. That's it!

## First Time Setup

1. **Launch** the application
2. **Select** your Downloads folder (already selected by default)
3. **Click "Analyze"** to see what will be organized
4. **Review** the preview table
5. **Click "Organize"** to sort your files

## What Gets Organized?

### Downloads Profile (Default)

Your files will be sorted into these folders:

- **Documents** - PDFs, Word, Excel, PowerPoint files
- **Images** - Photos, pictures, graphics (JPG, PNG, etc.)
- **Videos** - Movies, clips, recordings (MP4, AVI, etc.)
- **Audio** - Music, podcasts (MP3, WAV, etc.)
- **Archives** - ZIP, RAR, 7Z compressed files
- **Installers** - EXE, MSI setup files
- **Code** - Programming files (Python, JavaScript, etc.)
