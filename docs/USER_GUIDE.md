# AutoFolder AI - User Guide

## Welcome to AutoFolder AI

AutoFolder AI is a smart file organizer for Windows that automatically categorizes and organizes your files using intelligent rules and optional AI classification.

## Getting Started

### First Launch

1. Double-click `AutoFolder_AI.exe` to launch
2. The default folder is set to your Downloads folder
3. Click "Browse..." to select a different folder
4. Click "Analyze" to preview the organization

## Organization Profiles

AutoFolder AI comes with several built-in profiles:

### Downloads Profile (Recommended for most users)

Perfect for organizing photos and videos:
- **Photos**: All image formats

For gamers:
- **Recordings**: Gameplay footage

For professionals:

AutoFolder AI handles duplicate filenames smartly:
- **Rename**: Adds (1), (2), etc. to duplicates
- **Skip**: Leaves original file in place
- **Overwrite**: Replaces existing file (optional)

## Troubleshooting

### Files Not Being Organized

**Possible causes:**
- Files don't match any rules
- Files are hidden (disabled by default)
- Files are in use by another program

**Solution:**
- Check file extensions match profile rules
- Close programs using files
- Try custom rules

### "Permission Denied" Error

**Cause:** Windows file permissions

**Solution:**
- Run as Administrator
- Check folder isn't system-protected
- Close programs using those files

### Undo Not Working

**Cause:** Target folder was modified after organization

**Solution:**
- Don't modify organized folders before undoing
- Check undo history is not full (max 10 operations)

### AI Features Not Available

**Cause:** AI dependencies not installed or disabled

**Solution:**
- Ensure you have Pro version
- Check "Use AI classification" is enabled
- First use requires model download
